		function getValues() {
			//Get values from the page
			let startValue = document.getElementById("startValue").value;
			let endValue = document.getElementById("endValue").value;

			//We need to validate our input
			//Parse into integer
			startValue = parseInt(startValue);
			endValue = parseInt(endValue);

			if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
				//Call generateNumbers
				let numbers = generateNumbers(startValue, endValue);
				//We call displayNumbers
				displayNumbers(numbers);
			} else {
				alert("You must enter an integer");
			}
		}

		//Generate numbers from startValue to the endValue
		//Logic function(s)
		function generateNumbers(sValue, eValue) {
			let numbers = [];

			//We want to get all numbers from start to end
			for (let index = sValue; index <= eValue; index++) {
				//This will execute in a loop until index = eValue
				numbers.push(index);
			}
			return numbers;
		}

LOOPS

Loops is an app that loops through a series of numbers based on the Start and End numeric values entered by a user. The code contains three functions.

Function: getValues()

  1. This function gets the starting and ending values from a user.
  2. The values are then parsed into an Integer, then validated to ensure they are numeric by using the Numeric property of the isInteger method.
  3. If the values are numeric, the generateNumbers() function is called.
  4. Then, displayNumbers() function is called.
  5. If not, an alert is generated with the message, "You must enter an integer.

Function: generateNumbers()

  1. This function is initiated with two arguments, sValue and eValue.
  2. Then, an empty array is initiated called numbers.
  3. Next, a for loop is is used to loop to evaluate the Start and End values from the user. Index is initialized to sValue in the first part of the for loop condition.
  4. In the second part of the for loop condition, the loop will run until index is <= eValue.
  5. Then, in the 3rd part of the for loop condition, index increments by 1 each time through the loop.
  6. The Block part of he funcion assigns the value of index to the numbers array each time through the loop.
  7. Finally, the numbers array is returned abd passed to the displayNumbers() function.

Function: displayNumbers()

  1. The job of this function is to display the numbers on the screen according teh the specifications.
  2. The function is implemented with an argument. The argument takes the numbers array which was returned via the generateNumbers() function.
  3. The variable templateRows is initiated with an empty string. This will be used to write out each row of numbers to the page in a table.
  4. A for loop is used to loop through the numbers array and then assign each individual number to a variable called number.
  5. Each individual number from the numbers array is assigned to templateRows using a template literal.
  6. Likewise, to BOLD each even number per the specification, a className variable is created and assigned the string, "even".
  7. To identify the even numbers, each individual number is divided by 2 using the modulus operator which returns a remainder.
  8. If the remainder equals 0, "even" is assigned is assigned to className, else, "odd" is assigned.
  9. The variable className is also assigned to templateRows using a template literal.Specifically, it is assigned as a class in the tag of the table row.
  10. In the site.css file, the font-weight for the odd and even classes are styled appropriately: even is bolded and odd is set to 100 or normal.
  11. Finally, to write out the table in the HTML document, the getElementById property of the DOM is used to populate the innerHTML identified by the id "results" with templateRows. This is the variable containing all the individual numbers along with the class name of odd or even so they can be bolded or not.
